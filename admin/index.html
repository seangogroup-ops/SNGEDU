Câu hỏi trong đề đang chọn

Chọn một môn học và một chương/đề ở thanh bên trái để xem câu hỏi.

Thêm 1 câu hỏi

Câu hỏi sẽ được lưu vào môn và chương đang chọn ở thanh bên trái.

Số thứ tự: để trống = tự động xếp cuối danh sách; nhập số để chèn câu hỏi vào đúng vị trí đó (các câu phía sau tự dời xuống).

Bấm nút tròn cạnh đáp án đúng — thẻ tương ứng sẽ tự chuyển sang màu xanh.

Dán nhiều câu hỏi cùng lúc

Định dạng: Câu 1: Nội dung..., các dòng tiếp theo A. đáp án ... D. đáp án, đánh dấu đáp án đúng bằng dấu * phía trước (vd *B. đáp án), dòng Gt: giải thích không bắt buộc. Mỗi câu cách nhau 1 dòng trống.
hoặc tải file lên

Bấm để chọn file, hoặc kéo-thả file vào đây
Tự động nhận diện và trích câu hỏi/đáp án từ file quiz .html hoặc file .txt/.md theo đúng định dạng ở trên. Nội dung trích được sẽ tự động thêm vào ô phía trên.

Xuất câu hỏi ra file (.txt)

Tải câu hỏi về máy dưới dạng file văn bản, dùng để lưu trữ, chia sẻ hoặc nạp lại qua tab "Dán nhiều câu hỏi".
